Colored text Generator

For Minecraft: Bedrock Edition 1.18+

     
   

Don't forget to use Glow Ink Sac on sign! 🛈 After Minecraft Bedrock 1.17 update, any color that you could set through the symbol "§" became dull.

Solution: Use any dye (even if you specified a different color through "§"), and use "Glow Ink Sac" on the sign.

NOTE: In the versions 1.16.220 - 1.16.221, mechanic of using Glow Ink Sac already implemented, but since "Glow Ink Sac" was not added, make signs bright became IMPOSSIBLE.
